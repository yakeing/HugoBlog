防止网站文章被采集

禁止F12 禁止Ctrl+s 禁止ctrl+u 禁止选中文字 禁止鼠标右键

作为一名SEOer，相信大家都应该听说过“文章采集”，自己原创手写的文章半小时内被别人原模原样采集走了，最可气的是自己发布的文章没有排名，甚至是还没有被百度搜索引擎收录，而那个采集我们文章的人却已经拥有了一个很不错的排名，相当于我们在给别人做嫁衣，大家是不是看到这种采集的人就很头痛。

采集分为两种，一种是机器/程序采集，另一种是人工手动采集，市面上常见的采集工具（如：火车头、八爪鱼）是常见的一种采集程序，要采集成千上万的数据必须要用到采集工具。但是也有一些小站把原创和采集相结合，30%原创+70%采集，而这70%的采集一般都是去目标网站里面手动复制文章然后进行排版，手动采集文章主要是选择文字复制粘贴、F12审查元素、Ctrl+U查看源代码、Ctrl+S另存为，今天博主就教大家怎么使用JavaScript来防止他人通过这几种方法采集你的文章。

利用JavaScript禁止ctrl+u、ctrl+s、f12、选中文字

把的这些代码添加在网站页面的 <body><\body> 之间

禁止选中文字

//网站禁止选择文字
document.body.onselectstart = function() {
  self.event.returnValue=false
};

禁止鼠标右键

//网站禁止右键
document.body.oncontextmenu = function() {
  self.event.returnValue=false
};

禁止F12审查元素

  //按键触发
document.onkeydown = function(){
  //禁止 F12
  if (window.event && window.event.keyCode == 123) {
    event.keyCode = 0;
    event.returnValue = false;
  }
}

禁止Ctrl+S网页另存为

//按键触发
document.onkeydown = function(){
  //禁止ctrl+s
  if (event.ctrlKey && window.event.keyCode==83){
    return false;
  }
}

禁止F5刷新页面

//按键触发
document.onkeydown = function(){
  //禁止 F5
  if (window.event && window.event.keyCode == 116) {
    event.keyCode = 0;
    event.returnValue = false;
  }
}

如果想同时实现以上的几种效果，可以把上面几段代码放在同一个 <script></script> 里面，如下：

//禁止人工采集
document.onkeydown = function(){
  //禁止ctrl+u
  if (event.ctrlKey && window.event.keyCode==85){
    return false;
  }
  //禁止 F12
  if (window.event && window.event.keyCode == 123) {
    event.keyCode = 0;
    event.returnValue = false;
  }
  //禁止ctrl+s
  if (event.ctrlKey && window.event.keyCode==83){
    return false;
  }
  //禁止 F5
  if (window.event && window.event.keyCode == 116) {
    event.keyCode = 0;
    event.returnValue = false;
  }
}

//网站禁止右键
document.body.oncontextmenu = function() {
  self.event.returnValue = false
};
//网站禁止选择文字
document.body.onselectstart = function() {
  self.event.returnValue = false
};

利用纯代码禁止选中文字与鼠标右键

降以下代码添加到网站主题下header.php文件中的 <body> 中即可实现禁止选中文字与鼠标右键。

oncontextmenu="return false" onselectstart="return false"

<body <?php body_class()?> oncontextmenu="return false" onselectstart="return false">

以上利用JavaScript实现禁用的方法简单来说就是当用户使用键盘执行某一命令是返回的一种状态，而这种状态对应的则是keyCode(键码值)，键盘上的每一个按键都会有一个对应的keyCode，大家可以根据自己的需求设置不同的禁止命令。

说在最后: 上面的这些做法仅仅是对网站做了一层简单的防护，懂编程的人可以通过禁用JavaScript继续采集，但是应付一般的小白来说应该足够了。
