GitHub

GitHub是一个面向开源及私有软件项目的托管平台，因为只支持git 作为唯一的版本库格式进行托管，故名GitHub。

GitHub于2008年4月10日正式上线，除了Git代码仓库托管及基本的 Web管理界面以外，还提供了订阅、讨论组、文本渲染、在线文件编辑器、协作图谱（报表）、代码片段分享（Gist）等功能。目前，其注册用户已经超过350万，托管版本数量也是非常之多，其中不乏知名开源项目 Ruby on Rails、jQuery、python 等。

Hugo

作者 史蒂夫·弗朗西亚 (Steve Francia) google的产品负责人 现在美国纽约市

Hugo是由Go语言实现的静态网站生成器。简单、易用、高效、易扩展、快速部署。

Highlight.js

由 伊万·萨加拉耶夫(Ivan Sagalaev) 杰里米·赫尔(eremy Hull) 等人进行维护。

Highlight.js是用JavaScript编写的语法突出显示工具。它在浏览器和服务器上均可使用。它几乎可以与任何标记一起使用，不依赖于任何框架，并且具有自动语言检测功能。 Github

Pure.CSS

美国雅虎公司(Yahoo)

是一组轻量级、响应式纯CSS模块，适用于任何Web项目。

Font Awesome

由 戴夫·甘迪(Dave Gandy) 特拉维斯大通(Travis Chase) 耶利米切(Geremia Taglialatela) 布赖恩·塔尔博特(Brian Talbot) 共同维护该项目

一套绝佳的图标字体库和CSS框架，Font Awesome 字体为您提供可缩放矢量图标,它可以被定制大小、颜色、阴影以及任何可以用CSS的样式。

  • 点击 图标 进行预览矢量图标 Github
